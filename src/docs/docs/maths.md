# Categories

## Related commands

* `newcategory`(4): Shorthand to define new categories. For instance : `\newcommand{\sSet}{s}{S}{et}`, or `\newcategory{\Grp}{}{G}{rp}`
* `op`: The op exponent for dual categories.
